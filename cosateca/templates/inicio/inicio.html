La Cosateca

La Cosateca es un proyecto realizado como TFG para el grado de Ingeniería Informática - Ingeniería del Software de la Universidad de Sevilla. Ha sido llevado a cabo por el alumno Alfonso González Fernández.

Consiste en una aplicación web, usando Django como framework base, una base de datos PostgreSQL y HTML, CSS y Bootstrap como tecnologías, aunque también se usará algo de JavaScript para algunas funciones extras.

Para ver todo el código fuente, dejo el link al repositorio de GitHub aquí.

En la Cosateca podrás:
  • Ofrecer objetos que no uses a otros
  • Pedir objetos que necesites
  • Comentar en las ofertas
  • Consultar el historial de préstamos

El objetivo es crear una comunidad que vaya en aumento en el que los componentes sean personas solidarias con ganas de ayudar a la persona que tengas al lado.

Tendrás que crearte una cuenta para poder interactuar con los demás usuarios.
